# VSingerXrossPlayer

「歌ってみた」動画を楽しむための、PSP XMB（クロスメディアバー）風Webアプリケーションです。
歌い手や楽曲の特徴をAI分析（現在はモックデータ）により可視化し、新しい視聴体験を提供します。

## 特徴

- **XMB風ナビゲーション**: 矢印キーによる直感的な操作（上下左右）。
- **AI特徴可視化**: 歌い手や楽曲の「Energy」「Mood」「Vocal」「Instrumental」をレーダーチャートで表示。
- **背景動画再生**: 選択中の楽曲を背景でループ再生。

## 動作環境

- Node.js (v18以上推奨)
- npm

## 起動方法

### 1. インストール

プロジェクトのルートディレクトリで以下のコマンドを実行し、依存関係をインストールします。

```bash
npm install
```

### 2. 開発サーバーの起動

以下のコマンドでローカル開発サーバーを起動します。

```bash
npm run dev
```

ブラウザで `http://localhost:5173` (または表示されるURL) にアクセスしてください。

### 3. ビルド

本番用にビルドする場合は以下のコマンドを実行します。

```bash
npm run build
```

生成されたファイルは `dist` ディレクトリに出力されます。

## 操作方法

- **↑ / ↓**: アイテム（歌い手、楽曲など）の選択
- **← / →**: カテゴリ（Singers, Songs, Settings）の切り替え
